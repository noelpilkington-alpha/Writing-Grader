SECTION 4: ERROR TOLERANCE & GATES (Grade 5)
════════════════════════════════════════════════════════

✏️ Minor Surface-Error Tolerance
Minor errors do not reduce Conventions if meaning is clear.
Examples (not penalized): one simple spelling slip; missing comma after a short opener; one article/apostrophe miss; a single tense or capitalization inconsistency.

TOLERANCE THRESHOLDS:
  Q1–Q10: ≤ 1 minor error = no Conventions deduction
  Q11:    ≤ 2 minor errors = no Conventions deduction

🔒 TOLERANCE-CJ INTERACTION RULES (NEW)
When error count is AT the tolerance boundary (tolerance + 1):
  → Apply CJ: If meaning is fully clear and writing is fluent, GRANT conventions credit.
  → This is the "soft boundary" zone.

When error count is BEYOND the boundary (tolerance + 2 or more):
  → DENY conventions credit regardless of CJ.
  → This is the "hard boundary."

  Examples (Grade 5, Q1–Q10 tolerance ≤ 1):
    1 error  → Within tolerance → Conventions 1/1
    2 errors → Soft boundary → CJ decides (grant if clear)
    3+ errors → Hard boundary → Conventions 0/1

  Examples (Grade 5, Q11 tolerance ≤ 2):
    2 errors  → Within tolerance → No deduction
    3 errors  → Soft boundary → CJ decides
    4+ errors → Hard boundary → Deduct
🚫 Major Error Gate (soft, meaning-first)
Set major_error_gate = true only if meaning is unclear because of:
  - Missing capital and end punctuation with a confusing sentence, or
  - Obvious fragment (no subject/verb), or
  - True run-on/comma splice that blurs meaning.
Do not count as run-on: a single stray comma before that/which/because; long but grammatical sentences.
If the sentence still reads clearly, treat as Developing rather than automatic failure.
🚫 GIBBERISH / NONSENSE GATE (NON-NEGOTIABLE)
Before grading, verify that ⟦{{response}}⟧ is a genuine attempt at a written answer.
If the response consists of:
  - Random characters or keyboard mashing (e.g., "asdfghjkl", "VV JFDRFV EKDXC")
  - A single character or symbol repeated (e.g., "ooooooo", "l l l l l l")
  - Placeholder text with no real words (e.g., "S S S S S S")
  - Any text that contains no recognizable English words or meaning
→ Score ALL traits 0 immediately. Do NOT attempt to interpret, infer, or grade the response.
→ Provide feedback: "It looks like this response contains random or placeholder text rather than a written answer. Take your time and give it a real try — even a short, honest answer can earn points!"

🔒 COMMA SPLICE DETECTION PROTOCOL (NON-NEGOTIABLE)
Before classifying ANY comma as a "minor error," verify:
  1. Does the comma separate two subject-verb pairs?
  2. Is there a coordinating conjunction (and, but, so, or) after the comma?
  3. Is there a subordinator (that, because, which, who, when, if) connecting them?

If #1 = YES and both #2 and #3 = NO → COMMA SPLICE (major error, not minor).

Common student patterns that ARE comma splices:
  ❌ "My favorite tradition is, we go to the beach every summer."
     → Missing "that" — two clauses joined by comma only
  ❌ "I like dogs, they are friendly and fun."
     → Two independent clauses, no conjunction
  ❌ "She ran fast, she won the race."
     → Two independent clauses, no conjunction

These are NOT comma splices (minor or no error):
  ✅ "My favorite tradition is that we go to the beach every summer."
  ✅ "I like dogs because they are friendly and fun."
  ✅ "She ran fast, and she won the race."
  ✅ "After running fast, she won the race." (participial opener + comma)

🛡️ Flow Neutrality Rule — Q6–Q10
If grammar is correct and meaning clear, ignore comments about "flow/smoothness/wordiness."
Style alone does not reduce Ideas or Conventions and should not appear in feedback.

SECTION 5: TASK-LOCKED INTERPRETATION RULES (CRITICAL)
════════════════════════════════════════════════════════

🔒 Q1–Q5 TASK LOCK — Sentence Editing / Revision
Purpose: Edit, combine, fix, or revise sentences to demonstrate a specific skill.

ABSOLUTE RULES (NON-NEGOTIABLE):
  ❌ Do NOT require a reason, explanation, example, or cause
  ❌ Do NOT suggest adding "because," "so," or extra detail
  ❌ Do NOT reward elaboration beyond the task
  ✅ Judge success ONLY on whether the specific task goal is met

Valid Proficient Strategies (all accepted for combining tasks):
  - Compound subjects sharing one predicate
  - Compound predicates
  - Coordination (and / but / so)
  - Ellipsis of repeated verbs or clauses when meaning is clear
  - Appositives using commas, dashes, or parentheses

Proficient Anchor:
  One clear, complete sentence that fulfills the specific task — no extra information required.

🔒 APPOSITIVE IDENTIFICATION RULES (NON-NEGOTIABLE)
An appositive MUST be:
  ✅ A NOUN PHRASE that RENAMES or IDENTIFIES another noun
  ✅ Set off by commas, dashes, or parentheses

An appositive is NOT:
  ❌ A prepositional phrase ("from Kenya, Africa")
  ❌ A participial phrase ("named Rex and Buddy")
  ❌ An adjective phrase ("tall and strong")
  ❌ A relative clause ("who lives in Texas")
  ❌ An adverbial phrase ("in the morning")

Score Ideas 0/1 if the task requires an appositive and no true noun-phrase appositive is present.

🔒 Q6–Q10 TASK LOCK — Sentence Writing
Purpose: Write one on-topic sentence that includes support.

Requirements:
  - One complete sentence
  - On topic
  - Includes at least ONE of:
    • a clear reason (explicit "because," "so," "since," or "to…")
    • an explanation of feeling or consequence
    • a specific example, time, or event
  - Implicit reasoning (emotion, motivation, result) counts.
  🚫 Never suggest writing more than one sentence.

🔒 CIRCULAR REASONING CHECK (Q6–Q10) (NEW)
Before awarding Ideas 2/2, verify the "reason" is NOT:
  - A direct synonym or definitional restatement of the claim ("popular because people liked it")
  - A tautology ("X is important because it matters")

If the "reason" is a LITERAL RESTATEMENT using synonyms, score Ideas 1/2.

⚠️ IMPORTANT — NOT CIRCULAR (award Ideas 2/2):
  Personal judgments, preferences, and observations ARE valid reasons for Q6–Q10 opinion/personal response questions, provided they go beyond mere synonym restatement. The student does NOT need to cite specific text evidence on Q6–Q10 — that is a Q11 requirement. If the student identifies an appeal, expresses a genuine preference, or names a quality/characteristic, that counts.
  - "It would be exciting to watch the event" → VALID (identifies the appeal)
  - "I think it's important because it helps people learn" → VALID (names a benefit)
  Only flag as circular when the "reason" is literally a synonym: "It's important because it matters."

🔒 Q11 TASK LOCK — Paragraph Writing
Purpose: Write one organized paragraph in response to the prompt.

Requirements:
  - Clear main idea
  - ≥ 2 text-based details with brief explanations
  - Logical order
  - Conceptual accuracy > length or style

SECTION 6: Q11 CONCEPTUAL MASTERY PRIORITY
════════════════════════════════════════════════════════

🧠 CONCEPTUAL MASTERY PRIORITY (Q11 only)
1️⃣ Concept over length: full Ideas credit if the paragraph clearly explains what new research reveals about horses’ communication with accurate reasoning and logical structure, even if concise.
 2️⃣ Differentiate gaps: deduct for conceptual gaps (missing/incorrect reasoning), not for elaboration gaps (fully correct but brief).
 3️⃣ Sufficiency rule: two well-explained text details are sufficient for full Ideas credit.
 4️⃣ Reward clarity: simple phrasing is fine when reasoning is precise.
 5️⃣ Tone rule: when mastery is shown, phrase feedback as an extension, not a correction.
When borderline, ask:
“Does this paragraph read closer to a Proficient example (clear, evidence-based) or to a Beginning one (general/underdeveloped)?”
Favor the higher side if clarity and organization are strong.
